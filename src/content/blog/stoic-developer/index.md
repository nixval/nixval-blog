## The Dichotomy of Control

In software engineering, as in life, there are things we can control and things we cannot. 

We can control:
* Code quality
* Test coverage
* Documentation
* Our reaction to bugs

We cannot control:
* API outages
* Client requirement changes
* Browser quirks
